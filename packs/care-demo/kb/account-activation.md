# Activating a new account

When a new account is created for you, an activation email is sent with a
one-time activation code and a link. You will need: a working email
inbox you control, the activation code from the email, and a strong
network connection.

## Steps

1. Open the activation link in the email. If the link expired, use the
   **Re-send activation code** option on the sign-in page.
2. Confirm your identity: enter the activation code, then set a password
   (12+ characters, mix of letters and digits).
3. Choose your default plan when prompted. Most customers see two
   options: *Standard* and *Premium*. Pick the one that matches what
   you ordered.
4. Wait for activation to complete. This usually takes under two
   minutes.

If activation does not complete within ten minutes, sign out and back in.
If that does not resolve it, the activation code may have already been
consumed by a previous attempt — contact support to issue a fresh code.

Deactivating an account does **not** cancel any active subscriptions; it
only locks sign-in. Cancellation is a separate request.
