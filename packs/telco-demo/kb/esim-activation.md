# Activating an eSIM

An eSIM lets you connect a new device to the Acme Mobile network without
inserting a physical SIM card. You will need: a compatible device, a
strong Wi-Fi connection, and the activation code that was emailed to you
when the line was provisioned.

## Steps

1. Open **Settings -> Cellular -> Add eSIM** on the new device.
2. Choose **Use QR code** and scan the QR image from the activation email.
   If the camera cannot scan it, choose **Enter details manually** and
   paste the activation code from the same email.
3. Confirm the cellular plan label when prompted. Most customers see two
   options: *Primary* (voice + data) and *Data only*. Pick the one that
   matches what you ordered.
4. Wait for the network to register. This usually takes under two
   minutes.

If the eSIM does not activate within ten minutes, restart the device. If
restarting does not resolve it, the activation code may have already been
consumed by a previous attempt — contact support to issue a fresh code.

Removing an eSIM does **not** cancel the line; it only frees the slot on
the device. Cancellation is a separate request.
